# Simple To‑Do List

This is my CS50 Final Project.

A simple command‑line To‑Do List application written in Python.

## Features
- Add tasks
- Show tasks
- Delete tasks
- Save tasks in a text file

## How to Run
python todo.py

## Files
- todo.py — main program
- tasks.txt — stores tasks
